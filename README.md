# 🐾 API Clínica Veterinária

API REST desenvolvida com Java e Spring Boot para gerenciamento de clientes e pets.

## 🚀 Tecnologias utilizadas
- Java
- Spring Boot
- MongoDB
- Maven

## 📌 Funcionalidades
- Cadastro de clientes
- Cadastro de pets
- Listagem de dados
- Atualização de informações
- Remoção de registros

## 🏗️ Estrutura do projeto
- Controller
- Service
- Repository
- DTO

## ▶️ Como rodar o projeto

1. Clonar o repositório:
git clone https://github.com/seu-usuario/api-clinica-veterinaria

2. Entrar na pasta:
cd api-clinica-veterinaria

3. Rodar o projeto:
./mvnw spring-boot:run

## 📬 Endpoints principais
- GET /clientes
- POST /clientes
- GET /pets
- POST /pets
